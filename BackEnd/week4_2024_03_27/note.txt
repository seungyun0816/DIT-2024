내장 객체 범위 (scope)

HTTP는 상태정보를 유지 할 수 없기 때문에
웹 상에서 상태정보를 유지하기위해 영역(scope)을 정하고 
영역간에 정보를 공유한다, 이 정보들은 유효한 사용범위가 정해져있다

아래 순서는 작은 범위에서 넓은 범위로 정렬 한 것
넓은 범위는 작은 범위를 둘 이상 포함 가능

page scope : 한 페이지에서만 유효
request scope : 요청에 의해 호출된 페이지와 요청이 전달 된 페이지까지 공유, 새로운 페이지를 요청하면 소멸
session scope : 세션이 유지되고 있는 동안 한 브라우저에서는 공유
application scope : web application 이 유지되는 한 유효
