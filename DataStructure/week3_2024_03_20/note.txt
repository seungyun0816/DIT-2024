Python문자열의 특징

  구간인덱싱

  nagative인덱싱
  [-]를 이용해 문자열의 뒤쪽 인덱스 값 부터 가져오는 것

  .upper(),.lower()
  각각 문자열을 대문자,소문자로 바꾸어서 반환한다
  이때 원본은 변경되지 않는다

  .strip()
  문자열의 시작,끝 부분의 여백을 제거하여 반환한다
  이때 원본은 변경되지 않는다

  .replace('a','b')
  문자열에 포함 된 a를 b로 치환한다

  .split('/')
  문자열에 포함 된 /를 기준으로 하나의 문자열을 여러 문자열로 분할한다

  조건문

  if~else문
    if 조건식 :
        ~
    else :
        ~
  if문,else문의 들여쓰기는 같은 위치에 있어여한다.

  pass
    if문에서 당장 처리 할 내용이 없다면 pass로 넘긴다

  in,not in
    sequene에 특정한 값이 들어있는지 확인 하고 ture,false를 출력
    'a' in b >>> b에 a가 포함되어있는지를 판별
  
    list의 경우 각 값 안에 들어있는 값은 판별하지 않는다.
    a in ['abc','def','ghi'] >>> false
    abc in ['abc','def','ghi'] >>> true

  함수
    def [함수명](매개변수): 형식으로 선언
    -호출에 의해 실행되는 코드
    -매개변수를 가질 수 있음
    -값을 반환 할 수 있음
  파이썬의 함수는 여러 값을 반환 가능
